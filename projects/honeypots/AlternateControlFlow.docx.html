Get relevant PCAP for machine from Network Logs:

PCAP = tshark(recentPacketCapture, IP)

Put RAMdump and PCAP in Same “Event Folder”

Run Magic Script(PID) on Directory:

listOfPIDs = volatility.ppid(PID) #use volatility to recursively find all related PID’s from original

volatilityOutput = volatility.all(listOfPIDs) #get all info from memory

listOfConnections = volatility.netstat(listOfPIDs) # find all related connections

pcap = tshark(listOfConnections) #get pcap related to connections

sharkOutput =tshark.all(listOfConnections, pcap) #get all info from wireshark

snortRules = snort.makeSet(volatilityOutput, sharkOutput) #make ruleset with output from previous functions

email(volatilityOutput, sharkOutPut, snortRules) #send formatted email with attachments to sys admin
